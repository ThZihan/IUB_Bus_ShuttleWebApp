About the Project

The IUB Smart Shuttle project aims to provide a seamless transportation experience for IUB students and staff, featuring real-time GPS tracking, booking, and notifications.

Key Features

Real-time GPS Tracking

Experience live updates on shuttle locations, accurate ETAs, and route visualizations for seamless travel.

Trip Scheduling
& Management

Plan trips in advance, receive notifications for upcoming schedules, and manage shuttle availability effortlessly.

Hotspot Integration

Stay connected with Wi-Fi hotspot availability on every shuttle for enhanced passenger convenience.

Passenger Authentication

Seamless boarding with NFC or RFID-based authentication using employee or student ID cards.

Occupancy Monitoring
& Reservation System

Reserve and select your seat through the app, complete with real-time availability updates.

Route Optimization

Advanced Algorithm based route optimization for faster and more efficient travel times.

Analytics & Reporting

Gain insights into passenger patterns, optimize routes, and access detailed performance reports for efficient management.

Sustainability Features

Reduce environmental impact with solar-powered tracking devices and eco-friendly operational designs.

Security & Data Privacy

Ensure secure data transmission with encryption and role-based access to sensitive features.

System Overview

The IUB Smart Shuttle system uses advanced hardware and software to provide real-time shuttle tracking and user notifications. It includes GPS, WiFi modules, RFID sensors, and a central cloud platform for data analysis.

Developer Team followed an Agile-Waterfall hybrid or Iterative Waterfall method
