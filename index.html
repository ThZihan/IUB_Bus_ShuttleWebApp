System Overview

The IUB Smart Shuttle system uses advanced hardware and software to provide real-time shuttle tracking and user notifications. It includes GPS, WiFi modules, RFID sensors, and a central cloud platform for data analysis.

Developer Team followed an Agile-Waterfall hybrid or Iterative Waterfall method
